はじめに Introduction
WACCA RATING ANALYZER
WACCA RATING ANALYZER は、WACCAのマイページで確認できる情報を元に、レーティングを上昇させる上で便利なデータを確認できるプログラムです。 The WACCA RATING ANALYZER provides you useful data for increasing your rating based on the data available on your WACCA My Page.
特徴 Features
データ収集なし No data collection
このプログラムはブラウザー上で動作し、あなたのプレイ成績がサーバーへ送信されることはありません。 This program runs in your browser and does not send your play results to the server.
レーティング候補譜面の上限数なし Show more 'Rating Candidate Charts'
レーティング候補譜面をはじめ、レーティングを上げるために便利なデータをすべての譜面で見ることができます。 You can see 'Rating Candidate Charts' and other useful data for increasing your rating on all charts.
レーティング上昇量が詳細に分かる Detailed rating increase
所定のスコアへ到達した際のレーティングの上昇量が、譜面ごとに分かります。 You can see the amount of increase in your rating when you reach a certain score for each chart.
注意 Caution
ブラウザーがプライベートモードである場合、タブを閉じると分析されたデータは失われます。なお再度プログラムを実行することで、同じ結果を得られます。 If your browser is in private mode, closing a tab will cause the analyzed data to be lost. But don't worry. Just run the program again and you will get the same result.
譜面定数はあくまで推測された値であるため、公式の値とは異なる可能性があります。 The chart constants are only guessed values and may be different from the official values.
このプログラムは暇つぶしを目的に作成されているため、品質は保証されておりません。役に立たなかったらごめんなさい。 The quality of this program is not guaranteed, as it was created as a way to pass the time. I'm sorry if this program did not help you...

譜面別レーティング Rating by Chart
定数 Constant
譜面ごとに内部的に設定されている、詳細なレベル値です。 This is the detailed level value that is assigned internally to each chart.
係数 Modifier
譜面のハイスコアに応じて定数に乗算される係数（補正値）です。 This is the modified value that is multiplied with a constant based on the high score of each chart.
レート （現在の譜面別レーティング） Rating (Current rating by chart)
定数を現在の係数（補正値）で乗じた値です。 This is the value of multiplied by the constant and the current modified value.
上限（最大の譜面別レーティング） Max (Max rating by chart)
定数を補正値の上限である「4.00」で乗じた値です。既に譜面別レーティングがこの値に到達している場合は、 MAX と表示されます。 This is the value of multiplying the constant by "4.00", the upper limit of the modified value. If rating by chart has already reached this value, MAX will be displayed.
チェックリスト Checklist
チェックリストでは、挑戦したいスコアボーダーを、譜面ごとに設定することができます。チェックリストに追加すると、それらを全て達成した場合のレーティングの上昇量を確認することができます。 The "Checklist" feature allows you to set the score borders you wish to be challenged with for each chart. As you add items to the checklist, you can see how much your rating would increase if you achieved them all.

キーワード: keywords:
単にテキストを入力すると、そのテキストを含むタイトルの譜面を検索します。 Simply enter text to search for charts with titles containing that text.
let は、曲名に「let」を含む曲を検索します。 let will search for songs that contain "let" in the song title.
big bang は、曲名に「big」および「bang」を含む曲を検索します。 big bang will search for songs that contain "big" and "bang" in the song title.
difficulty:
difficulty:expert は、難易度「EXPERT」を検索します。 difficulty:expert will search for charts of difficulty "EXPERT".
level:
level:12 は、レベル「12」の譜面を検索します。 level:12 will search for charts of level "12".
constant:
constant:13.4 は、譜面定数「13.4」の譜面を検索します。 constant:13.4 will search for charts of constant "13.4".
newer:
新曲枠であるか否かを検索します。trueまたはfalseを指定できます。 Searches for "newer charts", can be true or false.
newer:true は、「新曲枠」である譜面を検索します。 newer:true will search for charts of included in "Newer Charts".
newer:false は、「旧曲枠」である譜面を検索します。 newer:false will search for charts of included in "Older Charts".
半角スペースで区切ることで、複数のオプションを組み合わせることができます。 You can combine multiple options by separating them with a space.
level:14 stronger は、レベル「14」かつ曲名に「stronger」を含む譜面を検索します。 level:14 stronger will search for charts of level "14" and contain "stronger" in the song title.
